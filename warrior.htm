Warrior

The warrior is a frontline support class that has builds that focus on offensive buffing, cc, and damage. We run several builds for the warrior, because the class can be adjusted in several very meaninful ways for different purposes.

Sword Warhorn / Axe Shield Destroyer Build

Offensive Variant | Defensive Variant

This build has two variants, both both aim for a similiar purpose. The build plays similiar to a bulldog; the group sicks this build on a target, it sticks to the target. It's "bite" is a combination of CC and spike damage, and it has very strong cleave damage as well.

Power CC Greatsword / Hammer Build

Offensive Variant | Defensive Variant

This build has two variants, both both aim for a similiar purpose. The build focuses and heavy CC chains and damage, with some potent cleave and mobility through the weapon set. The build is a strong team-fight style build.

Power Support Hammer / Sword Warhorn Build

Berserker Variant

This build's aim is to combine some of the control and damage potential of Hammer with the additional mobility and utility of the Sword and Warhorn weapons. It is much more of a support build than the others.
